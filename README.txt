Esse projeto foi para testar um pouco das linguagens HTML e CSS. Futuramente penso em dar continuidade, melhorando a comunicação e interação com usuário, segurança e melhores praticas de programação. 

 A princípio, a ideia desse site é ser uma fabrica, que busca uma forma de divulgar seus produtos, o site será um catálogo. Contudo, irá ter o endereço e um email para contato.
